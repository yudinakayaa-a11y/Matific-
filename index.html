📖 Como Usar o Bookmarklet (Favorito)

💻 No Computador (Chrome/Edge):

Arraste o botão lá de cima diretamente para a sua barra de favoritos do navegador.

📱 No iPhone (Safari):
  • Adicione qualquer página aleatória aos seus favoritos.
  • Copie o código do botão (ou o texto do script).
  • Edite o favorito que você salvou: mude o nome para "Matific 5★" e apague o link dele, colando o script no lugar.
🤖 No Android (Chrome):
  • Adicione esta página atual aos favoritos do seu Chrome.
  • Edite o favorito, mude o nome para "Matific" e cole o código do script no lugar do link (URL).
  • Quando estiver jogando no Matific, digite "Matific" na barra de endereços e clique no favorito que aparecer na lista para ativar.
